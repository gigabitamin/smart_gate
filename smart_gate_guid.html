Smart Gate Monitoring System

아두이노와 Django를 활용한 지능형 차량 통제 시스템

프로젝트 개요 (Overview)

본 프로젝트는 저비용 하드웨어를 기반으로 지능형 차량 통제 및 실시간 데이터 모니터링 시스템을 구현하는 것을 목표로 합니다.

  • 개발 기간: 2025. 06. 10 ~ 2025. 06. 30
  • H/W: Arduino Mega 2560, HuskyLens, 초음파 센서, MPU6050, SHT31 등
  • S/W: Python (Django), C, MySQL, VSCode

핵심 기능 (Key Features)

  • 지능형 출입 통제: HuskyLens 태그 인식 기반, 허가된 차량만 자동 통과
  • 실시간 모니터링: 웹 대시보드를 통해 다중 센서 데이터 시각화
  • 데이터 기반 분석: 수집된 로그 기반 통계 제공 (차량 통행량, PASS/STOP 비율)
  • 원격 제어 및 안전 모드: 웹을 통한 원격 차단기 제어 및 시스템 비활성화

기술적 도전과 해결 (Challenges & Solutions)

센서 신호 충돌 및 오작동
Debounce 및 비동기 처리 로직을 구현하여 시스템 안정성 확보
다중 센서 데이터 통합 관리
각 센서 기능을 독립적 함수로 모듈화하여 유지보수성 및 확장성 향상
H/W와 웹 간의 통신 지연
시리얼 통신 프로토콜 최적화 및 Django REST API 구성으로 실시간 데이터 전송 구현

프로젝트 회고 (Reflection)

하드웨어 한계의 S/W적 극복: 비동기 처리의 중요성을 체감하고 소프트웨어 아키텍처로 하드웨어의 물리적 한계를 극복했습니다.

데이터 신뢰성과 엣지 컴퓨팅: 센서 노이즈를 다루며, 정확한 데이터 처리를 위한 엣지 단에서의 데이터 전처리의 필요성을 배웠습니다.

사용자 경험(UX)과 Fail-Safe 설계: 사용자 관점의 명확한 피드백과 시스템 고장 시 안전을 보장하는 Fail-Safe 설계의 핵심을 파악했습니다.

향후 목표 (Future Goals)

  • 무선 통신 적용: ESP32/8266 모듈을 추가하여 물리적 제약 없는 원격 제어 기능 구현
  • 분석 기능 고도화: 머신러닝을 활용한 특정 시간대 교통량 예측 모델 개발
  • 알림 기능 도입: 이상 데이터(충격, 고온 등) 감지 시 관리자에게 자동 알림 기능 추가
